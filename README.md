# RPG-Project
Installation instructions:

1. Make sure you have the latest versions of C++/SFML installed.
2. Pull this repo (or just copy the source code).
3. Switch to the directory and run make clean in your terminal to clear out any .o files and compile new ones.
4. Execute the game file by doing ./game in your terminal.
